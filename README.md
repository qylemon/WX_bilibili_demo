# 小程序入门Demo

### 项目名称
仿bilibili的小程序精简版demo

### 如何引入到自己的项目中使用
- 使用命令行,在命令行中输入以下命令将项目Clone至本机中,在微信开发者工具中导入项目即可
  > git clone git@github.com:qylemon/WX_bilibili_demo.git

- 点击项目右上角的 Clone or download ,选择 Download ZIP ,将文件下载到本地中,解压缩后使用微信开发者工具导入
